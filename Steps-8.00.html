Dataset links:

Quarter ending December 31, 2021 (as of 06/07/22)

Open these links and save the data on your device then:

Open power BI Desktop

  1. Click Get data from power Bi desktop
  2. Under Get data, select Excel workbook
  3. Select 211231 WIOA program spending csv file and click open
  4. In the navigator window select the following excel sheets: DW_Sum, Youth-Sum and Adult_Sum

       5. Click on the Load and Transform data button to load the data.


Follow the same steps to load Texas WIOA funding for counties and the count table

  1. in the navigator window, select MyData
  2. Select transform.

Follow the same steps to load US Census bureau regions and division

       1.  Select Table1 and click Load and Transform
